ADVANCED ENDPOINTS

The Advanced Endpoints collection provides specialized and more complex operations within the Upgrade.chat CRM system. These endpoints offer enhanced functionality for contact management and subscription handling, catering to more specific or advanced use cases.

Key features include:

  • Adding or updating contacts using an import method
  • Managing subscriptions without associated order or payment information

Add/Update Contact (Using Import Method)

Add or update a contact using the import method.

POST {environment domain}/api/services/CRM/Import/ImportContact

Request Parameters

FIELD NAME FORMAT VALID VALUES DESCRIPTION
star string Any string Likely a rating or importance indicator
importType string "Lead" Specifies the type of import
ignoreInvalidValues boolean true, false Whether to ignore invalid values during import
matchExisting boolean true, false Whether to match with existing records
overrideLists boolean true, false Whether to override existing lists
createUser boolean true, false Whether to create a new user
sendWelcomeEmail boolean true, false Whether to send a welcome email
welcomeEmailTemplateRef string Any string Reference to the welcome email template
contactId integer Any non-negative integer Unique identifier for the contact
contactXref string Any string Cross-reference identifier for the contact
userPassword string Any string Password for the user account
personalInfo object Complex object Contains personal information of the contact
businessInfo object Complex object Contains business-related information
propertyInfo object Complex object Contains property-related information
assignedUser string Any string User assigned to this contact/lead
followUpDate string (ISO 8601) Valid date-time Date and time for follow-up
notes string Any string Additional notes
dateCreated string (ISO 8601) Valid date-time Date and time of record creation
leadStageName string Any string Name of the lead stage
leadSource string Any string Source of the lead
leadDealAmount number Any number Amount associated with the lead/deal
currencyId string Any string Identifier for the currency used
affiliateCode string Any string Code for affiliate tracking
campaignId string Any string Identifier for the associated campaign
channelId string Any string Identifier for the channel
classificationInfo object Complex object Contains classification details

Add Subscription(s) Without Order or Payment Information

Add or update a contact's subscription directly without first adding an invoice and payment.

POST {environment domain}/api/services/CRM/OrderSubscription/Update

Response Body

    { 
    ---  
    }

Sample Request

    curl -X 'POST' \
    'https://crm.upgrade.chat//api/services/CRM/OrderSubscription/Update' \
    -H 'accept: */*' \
    -H 'Content-Type: application/json;odata.metadata=minimal;odata.streaming=true' \
    -H 'Authorization: Bearer <your token>' \
    -d '{
    "connectedAccountId": "string"
    }'

Response: 200 (Success)
