COMMONLY USED ENDPOINTS

The following endpoints are involved in a variety of operations related to organizational units, events, contacts, pipelines, cases, invoices, and dictionary management within the Upgrade.chat platform.

CRM - Contacts & Leads

Endpoint Description
POST /api/services/CRM/Case/Delete Delete a case.
POST /api/services/CRM/Case/GetCases Get cases.
GET /api/services/CRM/Contact/GetContactInfo Get contact information.
POST /api/services/CRM/Contact/CreateOrUpdateContact Create or update a contact.
POST /api/services/CRM/Contact/DeleteContact Delete a contact.
POST /api/services/CRM/Contact/GetAllContacts Get all contacts.
POST /api/services/CRM/Contact/GetContactDetails Get contact details.
POST /api/services/CRM/Contact/GetContactGroups Get contact groups.
POST /api/services/CRM/Contact/SearchContacts Search contacts.
POST /api/services/CRM/Contact/UpdateContact Update a contact.
POST /api/services/CRM/Contact/GetAllowedAssignableUsers Get allowed assignable users.
POST /api/services/CRM/Contact/GetContactInfoForUser Get contact information for a user.
POST /api/services/CRM/Lead/Create Create a lead.
POST /api/services/CRM/Lead/DeleteLead Delete a lead.
POST /api/services/CRM/PersonContact/CreateUserForContact Create a user for a contact.
POST /api/services/CRM/OrganizationContact/CreateOrgUnitForOrganization Create an organization unit for an organization.
POST /api/services/CRM/OrganizationContact/GetOrganizations Get organizations.

CRM - Dictionary Definitions

Endpoint Description
POST /api/services/CRM/Dictionary/AddDictionaryValue Add a dictionary value.
POST /api/services/CRM/Dictionary/DeleteDictionaryValue Delete a dictionary value.
POST /api/services/CRM/Dictionary/GetDictionaryEntries Get dictionary entries.
POST /api/services/CRM/Dictionary/GetDictionaryValues Get dictionary values.
POST /api/services/CRM/Dictionary/GetTags Get tags.
POST /api/services/CRM/Dictionary/UpdateDictionaryValue Update a dictionary value.

CRM - Orders, Invoices and Payments

Endpoint Description
POST /api/services/CRM/OrderSubscription/Update Update an order subscription.
POST /api/services/CRM/Invoice/Create Create an invoice.
POST /api/services/CRM/Invoice/EmailInvoice Email an invoice.
POST /api/services/CRM/Invoice/GetInvoiceDetails Get invoice details.
POST /api/services/CRM/Invoice/GetInvoicePdf Get an invoice PDF.
POST /api/services/CRM/Invoice/GetInvoices Get invoices.
POST /api/services/CRM/Invoice/GetPaymentMethods Get payment methods.
POST /api/services/CRM/Invoice/PayInvoice Pay an invoice.
POST /api/services/CRM/Invoice/VoidBankCardPayment Void a bank card payment.
POST /api/services/CRM/OrderSubscription/GetSubscriptionHistory Get subscription history.
POST /api/services/CRM/OrderSubscription/Update Update an order subscription.
POST /api/services/CRM/ServiceProduct/GetAll Get all service products.

Platform - Event Subscriptions

Endpoint Description
POST /api/services/Platform/Event/GetSubscriptions Get event subscriptions.
POST /api/services/Platform/Event/Subscribe Subscribe to an event.
POST /api/services/Platform/Event/Unsubscribe Unsubscribe from an event.
POST /api/services/Platform/MemberSubscription/GetMemberInfo Get member information.
POST /api/services/Platform/OrganizationUnit/CreateOrganizationUnit Create an organization unit.
POST /api/services/Platform/User/AddToRole Add a user to a role.
POST /api/services/Platform/CommonLookup/FindUsers Find users.
POST /api/services/CRM/Case/GetCases Get cases.

Platform - Login Password Authentication

Endpoint Description
